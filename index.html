Welcome to the Interactive Web Map

This tool streamlines the collaborative application of the framework at the REST-COAST pilots.

Users of this tool can view, edit, or add polygons as Coastal Units at specific pilot sites.

New users of this tool can refer to the following documents for more detailed information:

 1. Deliverable 4.2.

 2. Scientific Article.

This tool complements the framework application data for any Coastal Unit.

 ➜ For existing Coastal Units, the users can directly work on the application data in a collaborative manner.

 ➜ When mapping a new Coastal Unit, the users can assign a template framework application sheet for collaboration.

📂 Users can access the existing and template data sheets through the shared directory.

This link ensures access to shared folders for all nine Rest-Coast pilot sites as well as new locations, each of which includes:

 ➜ the framework application data for Existing Coastal Units

 ➜ the template data sheets for New Coastal Units

IMPORTANT: If an existing Coastal Unit is modified (e.g. metadata, shape...), or a new Coastal Unit is identified (e.g. by drawing a polygon or loading a shape file):

 1.Use Save Coastal Units button to download the most recent version of the Coastal Units.

 Save using the default file name, which includes a date stamp for version control, i.e. CoastalUnits_YYYY-MM-DD_HH-MM.json.

 2.Upload this new version to the shared project directory at: upToDateCoastalUnits

 If access is denied, please contact cengiz.arslan@wur.nl for folder access permission.

Users can continue with the Interactive Web Map through:

 📍 Load Rest-Coast button to initialize the map with the Rest-Coast pilots.

 🗺 Load Map button to launch a blank map.
